COMMENT BLOCK MAKER

Utility for programmers, converting a text string to a 72 character wide comment block. Usage Instructions are available in the Help menu, and more details are in the comment block at the top of the script.

SYSTEM REQUIREMENTS

Python 2.6 or greater is required.

Tested with Python 2.7 and 3.4 on Debian GNU/Linux, and 2.6 on Mac OS X. MS Windows OS is supported, but has not been tested.
